# Skills Components

Components related to displaying and managing skills.

## Planned Components

- **SkillCard.vue** - Reusable skill card component
- **SkillDetail.vue** - Skill detail view component
- **SkillList.vue** - List of skills component
- **SkillGrid.vue** - Grid layout for skills
- **SkillNavigation.vue** - Previous/Next navigation between skills

## Usage

All components in this directory are auto-imported by Nuxt 4.
