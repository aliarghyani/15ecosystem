# Common Components

Shared UI components used across the application.

## Components

- **LanguageSwitcher.vue** - Language switching component for bilingual support
- **TopNav.vue** - Main navigation (to be created)
- **Footer.vue** - Site footer (to be created)
- **Breadcrumb.vue** - Breadcrumb navigation (to be created)

## Usage

All components in this directory are auto-imported by Nuxt 4. Use them directly without explicit imports:

```vue
<template>
  <LanguageSwitcher />
</template>
```
