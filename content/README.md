# Content Directory

**Note:** This directory is deprecated. Transcripts and summaries are now stored directly in `app/data/`.

## Current Structure

### Transcripts

**Location:** `app/data/transcripts/{locale}/`  
**Naming:** Use Persian video title as filename (e.g., `۷ عادت برای افزایش سلامت، قدرت و طول عمر.yml`)

### Summaries

**Location:** `app/data/summaries/{locale}/index.ts`  
**Method:** Add summaries manually to TypeScript files

- **Transcripts:** See `docs/how-to-add-transcripts.md`
- **Summaries:** See `docs/how-to-add-summaries.md`
